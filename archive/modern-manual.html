从笔记到可交付文档

适合知识库文章、方法论文档、产品说明与复盘报告。This style keeps a modern sans-serif voice while preserving long-form readability.

这是一套更现代的通栏长文模板。它不像传统书籍页面那样柔软，而是更接近知识库、产品文档和内部简报：标题清晰，段落稳定，列表和代码块有明确层级。

一、单栏不是单调

单栏页面依然可以有节奏。节奏来自标题间距、引用块、表格、代码块和提示框，而不是来自侧栏。When the body is uninterrupted, the reader can trust the page flow.

通栏长文的关键不是把内容拉满，而是在同一个版心里组织不同类型的信息。

适合 Markdown 的组件

  1. 二级标题负责章节推进。
  2. 三级标题负责局部说明。
  3. inline code 用于术语、路径、命令或字段名。
raw/        原始资料
wiki/       结构化知识
outputs/    交付产物
排版提示：如果文章里有大量清单或步骤，现代无衬线版本会比纯宋体版本更利于扫读。
层级视觉方式使用建议
H1大号粗体每页只出现一次
H2上边线 + 中号标题划分主要章节
Callout暖色浅底提示、风险、结论

这个版本更适合从 Markdown 自动生成 HTML，因为常见元素都有稳定样式，不依赖复杂结构，也不需要右侧注释栏。
